# Obesity Knowledge Base

A centralized repository for obesity clinical guidelines, peer-reviewed articles, and active clinical trials. Designed to support an AI-powered clinical decision support system.

## Structure

- **guidelines/** - Clinical practice guidelines from authoritative organizations
- **articles/** - Peer-reviewed research articles and meta-analyses
- **trials/** - Active and completed clinical trials data
- **crawler-config/** - Configuration files for web crawler and search criteria

## Quick Start

1. Add materials to their respective folders
2. Update crawler configuration in `crawler-config/`
3. System will automatically index and make searchable via the Q&A agent

## AI Agent Features

- Retrieves answers ONLY from materials on file
- Provides source citations
- Configurable search criteria and relevance rules
