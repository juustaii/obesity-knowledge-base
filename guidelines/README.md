# Clinical Guidelines

This folder contains official clinical practice guidelines from authoritative organizations on obesity management.

## Naming Convention

```
[ORGANIZATION]_[YEAR]_[TOPIC].pdf
```

## Example Files to Add

- WHO_2021_Obesity_Prevention_Treatment.pdf
- CDC_2023_Obesity_Management.pdf
- AMA_2022_Bariatric_Surgery_Guidelines.pdf
- AACE_2023_Weight_Management.pdf

## Metadata

Create a corresponding `.json` file with the same name containing:
- Title
- Organization
- Publication Year
- Key recommendations
- Target audience
