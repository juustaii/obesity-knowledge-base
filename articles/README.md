# Peer-Reviewed Articles

This folder contains peer-reviewed research articles, meta-analyses, and systematic reviews on obesity.

## Naming Convention

```
[YEAR]_[FIRST_AUTHOR]_[SHORT_TITLE].pdf
```

## Example Files to Add

- 2023_Smith_GLP1_Weight_Loss_Efficacy.pdf
- 2023_Johnson_Bariatric_Surgery_Outcomes.pdf
- 2022_Williams_Metabolic_Syndrome_Interventions.pdf
- 2023_Brown_Meta_Analysis_Obesity_Treatments.pdf

## Metadata

Create a corresponding `.json` file with:
- Full title
- All authors
- Journal name
- Publication year
- DOI
- Abstract
- Key findings
- Keywords
